# Multithreading-APIs
Projeto da Pós Go Expert para reproduzir chamadas em 2 apis distintas usando as goroutines, solicitadas pelas regras descritas no arquivo [Desafio](/desafio.txt).

## Como rodar
Abra um terminal, navegue até a pasta do projeto e execute: go run .

## Observações

- Certifique-se de ter o [Go](https://golang.org/dl/) instalado.
